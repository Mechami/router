Detailed Description

A router for passing messages to one-self.

Warning
Hubs currently do not have a system for cleaning dead routes introduced from sudden component destruction.
It is advised to remove routes manually within OnDestroy or OnDisable for the time being.
Todo:
Implement cleanup system for dead components.
