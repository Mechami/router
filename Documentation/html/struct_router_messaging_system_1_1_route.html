Detailed Description

Route struct for use with Router Messaging System.

Routes encapsulate the subscribing Component, the subscribed function and the subscribed event all in one object.

Multiple Routes can be used to subscribe a Component to multiple events at once.

Note
It is not advised to pass a multicast delegate as a RoutePointer.
